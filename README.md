# Employee Lifecycle Analytics & Attrition Prediction

1. Executive Summary
본 프로젝트는 금융권에서 대규모 고객 생애주기 분석 및 이탈 방어 시뮬레이션을 수행한 경험을 바탕으로, 
조직 내 핵심 인재의 생애주기(Employee Lifecycle)를 분석하고 퇴사 리스크를 선제적으로 예측하기 위해 
설계되었습니다.

2. Business Problem
핵심 인재의 이탈은 단순한 인력 공백을 넘어 조직의 지식 자산 유출과 대체 비용 증가로 이어집니다. 
사후 대응이 아닌, 데이터 기반의 조기 리스크 탐지(Early Warning) 체계가 필요합니다.

3. Analytical Approach

- Survival Analysis (생존 분석): 근속연수(Tenure)에 따른 퇴사 위험 구간(Hazard rate) 추정

- Explainable ML (설명 가능한 AI): Random Forest 및 SHAP Value를 활용하여, 
  개별 구성원의 퇴사에 영향을 미치는 핵심 요인(직무 만족도, 워라밸, 보상 등) 도출
